# LSTM Yelp Review Sentiment Regression

This project aims to **predict the sentiment scores of Yelp user reviews** using a **Long Short-Term Memory (LSTM)** based deep learning model.  
The model converts text data into numerical vectors and performs **regression** to estimate the positivity level of each review.

## Technologies Used

- **Python 3.11**
- **TensorFlow / Keras**
- **NumPy & Pandas**  
- **scikit-learn** 
- **Matplotlib**
